# Suggested Workflows

These workflows map to the custom agents and hooks in this repo.

## Plan -> Implement -> Review -> Release Notes

1. Use Planner to produce a scoped implementation plan.
2. Use the handoff button to start Implementation.
3. Handoff to Reviewer for risk and test coverage review.
4. Finish with Release Notes to summarize changes.

## Research -> Plan

1. Use Researcher to gather external references.
2. Handoff to Planner to integrate the findings.

## Diagnose failing tests

1. Ask Implementation to inspect failures and root cause.
2. Run targeted fixes, then rerun tests.
3. Handoff to Reviewer to confirm coverage.
